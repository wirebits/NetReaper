# NetReaper
A tool that deauth 2.4GHz Wi-Fi networks via ESP32 on serial console.

# Key Features
- Easily controlled by Serial Console.

# Hardware Requirements
- ESP32 Dev Development Board
- USB Data Cable with Data Transfer Support

> [!TIP]
> Use Linux system for better code management and compilation.

# Setup
1. Download VS Code from [here](https://code.visualstudio.com/download) according to system architecture.
2. Simply install it.
3. Open VS Code.
4. Click on `Extensions`.
5. Search `PlatformIO` and simply install it.
6. Click on `PlatformIO` button.
7. Wait for sometime to download necessary dependencies.
8. Done.

# Install
1. Download or Clone the Repository.
2. 3. Open VS Code.
4. Click on `Extensions`.
5. Search `PlatformIO` and simply install it.
6. Click on `PlatformIO` button.
7. Wait for sometime to download necessary dependencies.
8. Done.
3. Open the folder and just double click on `SpectraReaper.ino` file.
   - It opens in Arduino IDE.
4. Compile the code.
5. Select the correct board from the `Tools` → `Board` → `AmebaD ARM (32-bits) Boards`.
   - It is `Ai-Thinker BW16 (RTL8720DN)`.
6. Select the correct port number of that board.
7. Go to `Tools` → `Board` → `Auto Flash Mode` and select `Enable`.
8. Upload the code.
9. Done!
